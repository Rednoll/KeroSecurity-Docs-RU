Руководство¶

Kero-Security - библиотека для управления доступом к методам объектов.

Концепция¶

Для управления доступом, K-S создает прокси и ролевую конфигурацию к его методам.

Схема доступа

Набор всех правил, интерцепторов и т.д. для определенного типа.

Ролевая конфигурация

Более узкое представление схемы доступа заточеное под набор конкретных ролей. Представляет из себя “словарь” <Property, PreparedAction>, где подготовленные действия (запрет/разрешение доступа, вызов интерцептора и т.д.) заранее выбраны на основе набора ролей, для которого создавалась эта конфигурация.

Warning

В коде называется PreparedAccessConfiguration, автор так и не смог перевести это без тавтологий в тексте.
